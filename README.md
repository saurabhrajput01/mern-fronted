MERN Stack Student Management System

A simple MERN stack project for managing students with authentication and role-based access.

This project was built as an intern assignment to demonstrate backend APIs, authentication, and React integration.



Live Deployment

  Backend (Render)
  https://mern-backend-mzup.onrender.com
  
  Frontend (Netlify)
  Deployed using Netlify

 Features

- User Signup & Login
- JWT based authentication
- Password hashing using bcrypt
- Two roles: Admin and Student
- Admin dashboard to manage students
- Student dashboard to view and update profile
- Clean and simple UI

#Tech Stack

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Zod

Frontend
- React
- Vite
- Context API
- CSS
